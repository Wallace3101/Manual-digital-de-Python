PYTHON

Python es un lenguaje que apareció en 1991 quiere decir que tiene más de 30 años pero su utilización en diferentes campos ha ido en aumento situándolo en este 2022 como el lenguaje de programación más. Es un lenguaje un tanto intuitivo y es un leguaje muy directo. Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, conocido por su sintaxis clara y legible. Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Python es ampliamente utilizado en diversos campos como desarrollo web, ciencia de datos, inteligencia artificial, automatización de tareas y más.

  • Sintaxis Simple y Clara: Facilita la lectura y escritura del código.
  • Multiparadigma: Soporta programación orientada a objetos, funcional y procedimental.
  • Gran Ecosistema: Posee una vasta biblioteca estándar y una gran cantidad de bibliotecas de terceros.
  • Portabilidad: Funciona en múltiples plataformas como Windows, macOS y Linux.
  • Comunidad Activa: Gran cantidad de recursos, documentación y soporte comunitario.

Fundamentos de Estructuras de Datos y Operaciones en Programación

Print:

Es el encargado de encontrar todas las variables y mostrarlos. Puede llevar diferentes argumentos, que son separados por una coma, como por ejemplo: 

print(my_variable, my_int_variable, my_bool_variable, etc)
Consultar tipo de dato: 
print(type(variable))

Funciones del sistema usando print:

print(len(my_string_variable)): cuenta los datos del sistema Variables en una sola línea (mesclar variables): ¡No es recomendable! 

print(name, surname, alias, age = “William”, “Saravia” , “Willi”, 20)

Operadores aritméticos:

  • Adición (+): Sirve para sumar números y cadenas de texto, pero no entre tipos diferentes.
  • Sustracción (-): Sirve para sumar números.
  • Multiplicación (*): Sirve para multiplicar números.
  • División (/): Sirve para dividir números.
  • Módulos (%): El resto de una división.
  • Flor división (//): Aproximación de la división a un número entero.
  • Calcular un exponente (**): Es como multiplicar pero por dos.

Operadores comparativos:

Devuelve comparaciones de verdadero o falso para todos los números que se le introduzcan.

  • Mayor que (>)
  • Menor que (<)
  • Mayor o igual que (>=)
  • Menor o igual que (=<)
  • Igual (=)
  • Diferente (!=)

Operadores lógicos:

  • AND: Devuelve verdadero si ambos son verdaderos.
  • OR: Devuelve verdadero si uno de los dos es verdadero.
  • NOT: Devuelve verdadero si el valor es falso.

Formateo:

  • Concatenación: Se utiliza el signo + para unir cadenas de texto.
  • Interpolación: Se utiliza el signo % para insertar valores en una cadena de texto.
  • F-strings: Se utiliza el prefijo f para insertar valores en una cadena de texto.

Strings:

  • Indexación: Se utiliza para acceder a un carácter específico de una cadena de texto.
  • Slicing: Se utiliza para obtener una subcadena de una cadena de texto.
  • Métodos: Se utilizan para realizar operaciones con cadenas de texto.

Los Sting pueden llevar ciertos caracteres:

  • “\n”: Salto de línea.
  • “\t”: Tabulación.
  • “\”: Caracter de escape.

Funciones de los String:

  • .upper(): Convierte una cadena de texto a mayúsculas.
  • .lower(): Convierte una cadena de texto a minúsculas.
  • .capitalize(): Convierte la primera letra de una cadena de texto a mayúscula.
  • .title(): Convierte la primera letra de cada palabra de una cadena de texto a mayúscula.
  • .strip(): Elimina los espacios en blanco al principio y al final de una cadena de texto.
  • .replace(): Reemplaza una subcadena de una cadena de texto por otra.
  • .split(): Divide una cadena de texto en una lista de subcadenas.
  • .join(): Une una lista de cadenas de texto en una sola cadena.

Listas:

La lista ya vendría a ser una estructura, que nos sirve para estructurar datos. Una lista se define de la siguiente manera: 

my_list = list()    →   my_list = […]
my_other_list = []

Concatenación de listas:

print(my_list + my_other_list)

Métodos de las listas:

  • .append(): Añade un elemento al final de una lista.
  • .extend(): Añade los elementos de una lista a otra lista.
  • .insert(): Inserta un elemento en una posición específica de una lista.
  • .remove(): Elimina la primera aparición de un elemento de una lista.
  • .pop(): Elimina un elemento de una lista por su índice.
  • .index(): Devuelve el índice de la primera aparición de un elemento en una lista.
  • .count(): Devuelve el número de veces que un elemento aparece en una lista.
  • .sort(): Ordena los elementos de una lista.
  • .reverse(): Invierte los elementos de una lista.

Funciones de las listas:

  • print(len(my_list)): cuenta los datos de la lista.
  • print(my_list[0]): muestra el primer dato de la lista.
  • print(my_list[-1]): muestra el último dato de la lista.
  • print(my_list[1:3]): muestra los datos de la posición 1 a la 3.

Tuples:

Las tuples son una estructura de datos que se utilizan para almacenar colecciones de datos inmutables. Se definen de la siguiente manera: 

my_tuple = tuple()    →   my_tuple = (…)
my_other_tuple = ()

Concatenación de tuplas:

print(my_tuple + my_other_tuple)

Métodos de las tuplas:

  • .count(): Devuelve el número de veces que un elemento aparece en una tupla.
  • .index(): Devuelve el índice de la primera aparición de un elemento en una tupla.
  • tuples = list(): Convierte una tuple a lista, para asi poder manipular los datos.
  • del tuples: Borra la tuple. No se pueden borrar los datos uno por uno.

Funciones de las tuplas:

  • print(len(my_tuple)): cuenta los datos de la tupla
  • print(my_tuple[0]): muestra el primer dato de la tupla
  • print(my_tuple[-1]): muestra el último dato de la tupla
  • print(my_tuple[1:3]): muestra los datos de la posición 1 a la 3

Sets:

Los sets son una estructura de datos que se utilizan para almacenar colecciones de datos únicos. Se definen de la siguiente manera: 

my_set = set()
my_other_set = {…}

Métodos de los sets:

  • .add(): Añade un elemento a un set.
  • .remove(): Elimina un elemento de un set.
  • .discard(): Elimina un elemento de un set si está presente.
  • .pop(): Elimina un elemento de un set de forma aleatoria.
  • .clear(): Elimina todos los elementos de un set.
  • .union(): Devuelve la unión de dos sets.
  • .intersection(): Devuelve la intersección de dos sets.
  • .difference(): Devuelve la diferencia de dos sets.
  • symmetric_difference(): Devuelve la diferencia simétrica de dos sets.

Funciones de los sets:

  • print(len(my_set)): cuenta los datos del set
  • print(my_set): muestra los datos del set
  • print(int in my_set): Comprueba si un elemento int se encuentra en el set.
  • print(“str” in my_set): Comprueba si un elemento String se encuentra en el set.

Diccionario:

Los diccionarios son una estructura de datos que se utilizan para almacenar colecciones de datos en pares clave-valor. Se definen de la siguiente manera: 

my_dict = dict()
my_other_dict = {…}
Se puden definir de las siguientes maneras: 
                            
    Case 1:

        my_dict = “name”: “William”, “surname”: “Saravia”, “alias”: “Willi”, “age”: 20

    Case 2:
                            
        my_dict = {
            “Nombre”:“William”,
            “Apellido”:“Antonio”,
            “Edad”:35,
            “Lenguajes”:{“Python”,“Swift”,“Kotlin”}
        }
Donde {“Python”,“Swift”,“Kotlin”}; es un set.

Métodos de los diccionarios:

  • .get(): Devuelve el valor de una clave en un diccionario.
  • .pop(): Elimina un elemento de un diccionario por su clave.
  • .popitem(): Elimina el último elemento insertado en un diccionario.
  • .keys(): Devuelve una lista con las claves de un diccionario.
  • .values(): Devuelve una lista con los valores de un diccionario.
  • .items(): Devuelve una lista con los pares clave-valor de un diccionario.
  • .clear(): Elimina todos los elementos de un diccionario.
  • .update(): Actualiza un diccionario con los elementos de otro diccionario.

Funciones de los diccionarios:

  • print(len(my_dict)): cuenta los datos del diccionario
  • print(my_dict[“key”]): muestra el valor de la clave “key”
  • print(“key” in my_dict): Comprueba si una clave se encuentra en el diccionario.
  • my_dict[“Nombre”] = “Antonio” : Es para actualizar el valor que actualmente tiene “Nombre”. También sirve para añadirle un nuevo campo al diccionario, cambiando “Nombre”, por un nuevo campo y dándole un valor.
  • del my_dict[“Calle”] : Elimina elemento del diccionario. Una ves eliminado ya no se pueden recuperar.

Condicionales:

Los condicionales son estructuras de control que permiten tomar decisiones en un programa. Se definen de la siguiente manera: 

                                
        if condition:
        elif condition:
        else:

Operadores de comparación:

  • Mayor que (>): Devuelve verdadero si el operando de la izquierda es mayor que el de la derecha.
  • Menor que (<): Devuelve verdadero si el operando de la izquierda es menor que el de la derecha.
  • Mayor o igual que (>=): Devuelve verdadero si el operando de la izquierda es mayor o igual que el de la derecha.
  • Menor o igual que (=<): Devuelve verdadero si el operando de la izquierda es menor o igual que el de la derecha.
  • Igual (=): Devuelve verdadero si los operandos son iguales.
  • Diferente (!=): Devuelve verdadero si los operandos son diferentes.

Operadores lógicos:

  • AND: Devuelve verdadero si ambos son verdaderos.
  • OR: Devuelve verdadero si uno de los dos es verdadero.
  • NOT: Devuelve verdadero si el valor es falso.

Ejemplos:

Operadores de comparación
                                
            x = 5
            y = 10
            if x > y:
                print("x es mayor que y")
            elif x < y:
                print("x es menor que y")
            else:
                print("x es igual a y")
Operadores lógicos
                                
            a = True
            b = False
            if a and b:
                print("Ambos son verdaderos")
            elif a or b:
                print("Al menos uno es verdadero")
            else:
            print("Ambos son falsos")
Operador de negación
                                
            c = False
            if not c:
                print("c es falso")
            else:
                print("c es verdadero")

Bucles:

Los bucles son estructuras de control que permiten repetir un bloque de código varias veces. Se definen de la siguiente manera: 

                            
            for variable in iterable:
            while condition:

Bucle for:

El bucle for se utiliza para iterar sobre una secuencia de elementos. Se puede utilizar con listas, tuplas, diccionarios, sets y cadenas de texto. 

for i in range(5):
for i in range(1, 5):
for i in range(1, 10, 2):
es una estructura de control que permite iterar sobre una secuencia (como una lista, una tupla, un diccionario, un conjunto o una cadena) y ejecutar un bloque de código para cada elemento de esa secuencia. Es especialmente útil para procesar todos los elementos de una colección sin tener que gestionar manualmente el índice o la condición de terminación. Ejem.: 
                                
            frutas = ["manzana", "banana", "cereza"]
            for fruta in frutas:
                print(fruta)
Un for se va a repetir tantas veces como elementos tengamos en el parámetro definido, en este caso fruta tiene 3 elementos. A este bucle se le puede relacionar con list, tuples, set y dictionari. Esta iteración esta relacionada a conjuntos de valores. Pero también se le pude definir rangos y no necesariamente números de elementos: 
                                
            for i in range(5):
            print(i)
También puede trabajar con un bucle else y además, al igual que el bucle while a este se le puede añadir el break. 
                                
        Caso 1: // En el momento en que se rompe el for usando break, nos salimos directamente del bucle.
            for element in my_dict:
                print(element)
                if element == “Edad”:
                    break
        Caso 2:
            for element in my_dict:
                print(element)
            else:
                print(“El bucle para diccionario a finalizado”)

Bucle while:

El bucle while se utiliza para repetir un bloque de código mientras se cumpla una condición.

while condition:
Es una estructura de control de flujo que permite repetir un bloque de código mientras una condición especificada sea verdadera. Es útil para crear bucles que continúan ejecutándose hasta que la condición cambie a falsa. Ejem.: 

                                
            contador = 0
            while contador < 5:
                print("Contador:", contador)
                contador += 1
En este bucle debemos de buscar que la condición sea eventualmente falsa, porque si no este será un bucle infinito. También le podemos añadir un else, pero sin un if: 
                                
            contador = 0
            while contador < 5:
                print("Contador:", contador)
                contador += 1
            else:
                print(“Mi condición es mayor o igual que 5”)
También podemos añadir una condición en concreta y si además de eso queremos que cuando llegue a esa condición en concreta el bucle se detenga, añadiremos break 
                                
            contador = 0
            while True:
                print("Contador:", contador)
                contador += 1
                if contador >= 5:
                break

Funciones:

Las funciones son bloques de código que se pueden reutilizar en un programa. Se definen de la siguiente manera: 

def function_name(parameters):

Definición de funciones:

Las funciones son bloques de código que se pueden reutilizar en un programa. Se definen de la siguiente manera: 

                                
            def name ():   // Crear función
                print(“Esto es una funcion”)
            name ()   // Llamar a la función
En este caso, la función name () recibe un parámetro y muestra un mensaje de saludo con el parametro proporcionado. La función se llama con el argumento.

Parámetros:

También puede tener parámetros que definen con que es con lo que trabajara la función y la función aceptara tantos valores como los parámetros que se definió al momento de crear la función y no es necesario definir el tipo de variable solo es necesario que le metamos variables que puedan concatenar, pero si lo quieres definir lo haces al momento de crear una función: 
                                
            def sum_two_values (num1, num2): // Definir parámetros
                print(num1 + num2)
            sum_two_values (5, 7)  // Darles valores a los parámetros
Cabe recalcar que debemos de tener en cuenta que no con todas las variables se pueden realizar todas las funciones. La función además de poder recibir parámetros también puede retornarlos: 
                                
            def sum_two_values (num1, num2):
                return num1 + num2  // Valor que nos va a retornar
            my_result = sum_two_values (5, 7)  
            print(my_result)
Para interpolar los datos ingresado en variables String se estructura de la siguiente manera: 
                                
            def print_name (name, surname, alias = “Sin alias”):
                print(f“{name}  {surname}  {alias}”)
            print_name(“Leon”,“Willi”,“Wallace”) // Este alias remplaza al ya definido
En Python, puedes usar las f-strings (cadenas formateadas: f“{name} {surname} {alias}”) para insertar variables dentro de una cadena de texto. Las f-strings se crean colocando una 'f' antes de las comillas que delimitan la cadena y luego encerrando las variables entre llaves {}.
Cuando yo quiera darle o agregar cualquier tipo de datos se usa la siguiente sintaxis y prácticamente es como si yo le enviara elementos infinitos. Los parámetros se deben de separar por una coma: 
                                
            def print_text (*text)
                print(text)
            print_name(“Hola”,“Python”,“Back”)

Clases:

Las clases son plantillas que se utilizan para crear objetos en Python. Se definen de la siguiente manera: 

class ClassName:

Definición de clases:

Las clases son plantillas que se utilizan para crear objetos en Python. Se definen de la siguiente manera: 
                            
            class Person:
                def __init__(self, name, age):
                    self.name = name
                    self._age = age // Atributo privado
            my_person = Person(name, surname)

Funciones de las clases:

  • __init__(): Método especial que se llama cuando se crea un objeto de la clase.
  • self: Hace referencia al objeto actual.
  • __str__(): Método especial que se llama cuando se convierte un objeto a una cadena de texto.
  • __del__(): Método especial que se llama cuando se elimina un objeto de la memoria.
  • __repr__(): Método especial que se llama cuando se representa un objeto.
  • __len__(): Método especial que se llama cuando se obtiene la longitud de un objeto.
  • __getitem__(): Método especial que se llama cuando se obtiene un elemento de un objeto.
  • __setitem__(): Método especial que se llama cuando se establece un elemento de un objeto.
  • __delitem__(): Método especial que se llama cuando se elimina un elemento de un objeto.

Operaciones de las clases:

  • Instanciación: Crear un objeto de una clase.
  • Atributos: Variables que almacenan información sobre un objeto.
  • Métodos: Funciones que realizan operaciones con un objeto.
  • Herencia: Crear una nueva clase a partir de una clase existente.
  • Encapsulación: Ocultar los detalles de implementación de un objeto.
  • Polimorfismo: Capacidad de un objeto para tomar muchas formas.

Excepciones:

Las excepciones son errores que ocurren durante la ejecución de un programa. Se definen de la siguiente manera: 

try:
except Exception as e:

Definición de excepciones:

Las excepciones son errores que ocurren durante la ejecución de un programa. Se definen de la siguiente manera: 
                            
            try:
                result = 10 / 0
            except ZeroDivisionError as e:
                print("Error:", e)

Tipos de excepciones:

  • ZeroDivisionError: Error que ocurre al dividir un número por cero.
  • ValueError: Error que ocurre al pasar un argumento con un tipo incorrecto.
  • TypeError: Error que ocurre al realizar una operación con un tipo incorrecto.
  • IndexError: Error que ocurre al acceder a un índice fuera de rango.
  • KeyError: Error que ocurre al acceder a una clave inexistente en un diccionario.
  • FileNotFoundError: Error que ocurre al intentar abrir un archivo que no existe.
  • ImportError: Error que ocurre al importar un módulo que no existe.
  • ModuleNotFoundError: Error que ocurre al importar un módulo que no existe.
  • AttributeError: Error que ocurre al acceder a un atributo inexistente en un objeto.
  • Exception: Clase base para todas las excepciones en Python.

Manejo de excepciones:

  • try: Bloque de código que puede lanzar una excepción.
  • except: Bloque de código que se ejecuta si se lanza una excepción.
  • else: Bloque de código que se ejecuta si no se lanza una excepción.
  • finally: Bloque de código que se ejecuta siempre, independientemente de si se lanza una excepción.
Ejemplo: 
                            
            try:
                result = 10 / 0
            except ZeroDivisionError as e:
                print("Error:", e)
            else:
                print("La división se realizó correctamente")
            finally:
                print("Fin del programa")
La captura de excepción por tipo: 
                            
            try:
                print(num1 + num2)
                print(“No se ha producido un error”)
            except ValueError as error:
                print(error)

Módulos:

Los módulos son archivos que contienen funciones, clases y variables que se pueden reutilizar en un programa. Se definen de la siguiente manera: 

import module_name
from module_name import function_name

Definición de módulos:

Los módulos son archivos que contienen funciones, clases y variables que se pueden reutilizar en un programa. Se definen de la siguiente manera: 
                            
            import math
            print(math.pi)

Tipos de módulos:

  • Módulos estándar: Módulos que vienen incluidos con Python.
  • Módulos de terceros: Módulos que se instalan a través de un gestor de paquetes.
  • Módulos personalizados: Módulos creados por el usuario.

Importación de módulos:

  • import module_name: Importa todo el módulo.
  • from module_name import function_name: Importa una función específica del módulo.
  • from module_name import *: Importa todas las funciones del módulo.
  • import module_name as alias: Importa el módulo con un alias.

Uso de módulos:

  • import module_name as alias: Llama a una función de un módulo.
  • module_name.variable_name: Accede a una variable de un módulo.
